## [0.1.0] — 2026-06-08

首个可安装预览版：**AI 对话 + 笔记本** 学习桌面应用。

### AI 对话

- 对话 / 智能体 / 阅读 模式切换（Cursor 风格下拉）
- 多会话流式对话、历史会话页内切换（非侧栏分栏）
- 笔记 / 文档上下文 chip：发送后绑定到消息、可点击跳转
- TXT/MD 按当前章节加入对话；侧栏与 AI 面板开关持久化
- Skill 下拉菜单（设置页查看 / 导入 / 删除用户 Skill）
- AI 回复 / 历史对话加入笔记（含问答对与引用）
- 从 AI 笔记跳回对应对话会话

### 笔记本（compose）

- 多笔记本、文档解耦关联、条目层级与拖动排序
- 跨笔记本笔记树、文档角标、锚点跳转
- 笔记本 JSON 导出 / 导入
- `/` 命令与富文本笔记编辑器

### 阅读器

- TXT / MD / PDF / DOCX / 内置网页
- PDF 缩放与侧栏 hover 防护（page zoom 守卫）
- 文本选区 Ask AI

### 技术

- Electron + React + TypeScript；171 项单元测试
- Windows NSIS 安装包（`npm run dist:win`）
