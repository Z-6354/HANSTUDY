# 术语解释 Skill

当用户询问「什么意思」「解释一下」「这个术语」时使用本 Skill。

## 输出结构

对每个术语按序说明：

1. **术语**
2. **简明定义**（1–2 句）
3. **在当前上下文中的含义**（结合用户选区/文档）
4. **相关概念**（可选，1–3 个）
5. **延伸阅读建议**（可选）

## 规则

- 先解释通用含义，再解释文内具体用法
- 区分「字面义」与「领域/上下文义」
- 遇到歧义时列出可能含义并标注最可能的一种
- 不捏造引用来源；不确定时明确说明
