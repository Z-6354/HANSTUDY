# 文档摘要 Skill

当用户需要总结、概括、提炼要点时使用本 Skill。

## 输出结构

1. **一句话摘要**（≤ 80 字）
2. **核心论点**（3–5 条 bullet）
3. **关键细节**（可选，与论点对应）
4. **待进一步阅读**（如有遗漏或需原文确认处）

## 规则

- 优先基于用户提供的文档/选区，不编造未出现的信息
- 保留原文中的术语与专有名词
- 若内容不足，说明缺少什么信息
- 使用中文，条理清晰，避免空泛套话
